# Admin

Admin 是一个后台解决方案，它基于 Vue3 与 TDesign 实现。它可以帮助你快速搭建企业级中后台产品原型。

## 功能特性

- [x] 登录/注销
- [ ] 权限验证
  - [ ] 动态路由
  - [x] 权限指令
- [x] 实用组件
  - [x] ECharts
  - [ ] 上传组件
    - [ ] 普通上传
    - [ ] 端点续传
    - [ ] 头像上传（裁剪上传）
- [x] 控制台
- [x] 用户管理
- [x] 角色管理
- [ ] 表格
  - [ ] 基础表格
  - [ ] 内联编辑
- [x] 错误页面
- [x] CURD 模板生成器
